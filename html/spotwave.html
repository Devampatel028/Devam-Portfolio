Spot Wave Music Player

A Spotify-inspired music streaming application focused on delivering a rich audio experience directly in the browser, featuring playlist rendering, seek controls, and a fully responsive interface.

HTML5 CSS3 JavaScript
Live Demo Source Code

Overview

Spot Wave is a dedicated music player clone inspired by Spotify's iconic dark interface. The goal of this project was to understand and replicate complex media logic directly in vanilla JavaScript without relying on heavy frameworks.

The UI focuses on dark themes with vibrant green accents. Users can browse their library, play songs, pause, and seek through tracks seamlessly. It perfectly manages HTML5 Audio APIs under the hood while providing a completely custom visual player interface for volume and time control.

Key Features

✓ Responsive Spotify-inspired dark UI
✓ Dynamic playlist rendering and track selection
✓ Custom HTML5 Audio integration for seamless playback
✓ Interactive seek bar and volume controls
