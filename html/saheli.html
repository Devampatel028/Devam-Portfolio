Saheli — Smart PCOS Care

AI-powered healthcare platform with patient, doctor, and admin portals.

React Node.js MongoDB Express Gemini API Python
Live Source Code

Overview

"Saheli – Care for PCOS is a role-based healthcare platform that empowers women to detect and manage PCOS early. By combining AI insights, personalized wellness recommendations, and doctor consultations, Saheli helps women take control of their hormonal health."

It features a dedicated patient path for monitoring symptoms over time, a doctor dashboard for health professionals to manage their patient registry, and an admin console to manage the medical network.

Key Features

✓ Role-Based Access Control (Patient, Doctor, Admin)
✓ AI-Powered PCOS Risk Assessment via Gemini API
✓ Verified Doctor Dashboard for consultations
✓ Personalized Diet & Exercise Recommendations
